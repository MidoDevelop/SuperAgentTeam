## 输出内容

对每个 API 端点：

1. **基本信息**: 方法、路径、状态（新增/修改）
2. **请求**: Header、Path Params、Query Params、Request Body（含字段类型、是否必填、示例值）
3. **响应**: 成功响应（字段、类型、示例）、错误响应（错误码、消息）
4. **业务规则**: 权限要求、数据校验规则、特殊逻辑
5. **版本兼容**: 如果是修改现有 API，说明对旧客户端的影响

汇总输出一份 API 契约文档，写入指定的输出文件。
